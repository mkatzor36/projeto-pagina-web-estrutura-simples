Entendendo sobre o HTML5

  • A Base da Web Moderna

O HTML5 é a versão mais recente e mais poderosa da Linguagem de Marcação de Hipertexto (HTML). Ele representa um grande avanço em relação às versões anteriores, oferecendo uma série de novas funcionalidades e simplificando o desenvolvimento de páginas web.

O HTML5 é uma linguagem de marcação utilizada para estruturar e apresentar o conteúdo de páginas da web. Ele define a semântica e a estrutura de um documento, ou seja, como o conteúdo deve ser interpretado pelos navegadores.

    Características do HTML5:

  • Conteúdo multimídia: Permite incorporar vídeos, áudio e gráficos diretamente nas páginas web sem a necessidade de plugins adicionais;
  • Aplicações web offline: Possibilita o armazenamento de dados localmente no dispositivo do usuário, permitindo que aplicativos web funcionem mesmo sem conexão com a internet;
  • Gráficos 2D e 3D: Oferece APIs para criar gráficos e animações diretamente no navegador, eliminando a necessidade de plugins como o Flash;
  • Geolocalização: Permite que os sites acessem a localização geográfica do usuário, abrindo portas para novos tipos de aplicativos;
  • WebSockets: Facilita a comunicação bidirecional em tempo real entre o navegador e o servidor, tornando as aplicações web mais interativas;
  • Semântica: Fornece novas tags semânticas para descrever melhor o conteúdo da página, o que beneficia a otimização para mecanismos de busca (SEO) e a acessibilidade.

    Aplicações do HTML5:

  • Criação de sites responsivos: O HTML5, em conjunto com CSS3, permite criar sites que se adaptam a diferentes tamanhos de tela;
  • Desenvolvimento de aplicativos web: O HTML5, combinado com JavaScript e frameworks como React, Angular e Vue.js, é utilizado para criar aplicativos web complexos e interativos;
  • Criação de jogos: O HTML5 e o Canvas permitem criar jogos 2D e 3D diretamente no navegador.
  • Visualização de dados: O HTML5 pode ser utilizado para criar visualizações de dados interativas e personalizadas.

  • As Tags do HTML5

Imagine que você está construindo uma casa. As paredes, o teto, as janelas e as portas são os elementos básicos que compõem a estrutura da casa. No mundo da programação web, as tags HTML desempenham um papel semelhante. Elas são os elementos básicos que definem a estrutura e o conteúdo de uma página web.

As tags HTML são palavras-chave encerradas entre colchetes angulares (< e >). Cada tag indica ao navegador como interpretar o conteúdo que ela envolve.

    Tags para Títulos:

  • h1: Define o nível de título em uma página;
  • h2 até h6: Seguem a mesma lógica de h1, mas com menor importância visual.

    Tags para Parágrafos:

  • p: Define um parágrafo;
  • mark: Marca um trecho de texto como importante, geralmente com destaque visual;
  • small: Exibe o texto em um tamanho de fonte menor;
  • i: Exibe o texto em itálico;
  • u: Sublinha o texto;
  • strong: Exibe o texto em negrito;
  • ol: Cria uma lista ordenada;
  • ul: Cria uma lista não ordenada;
  • li: Define um item em uma lista;
  • blockquote: Cita um bloco de texto, geralmente de outra fonte;
  • font: Anteriormente usada para definir a fonte, cor e tamanho do texto, mas não é mais recomendada devido à flexibilidade do CSS;
  • del: Indica texto que foi deletado;
  • abbr: Define uma abreviação ou acrônimo.

    Tags para Links e Divisões:

  • a: Cria um hiperlink para outra página ou recurso;
  • hr: Insere uma linha horizontal para dividir seções.

    Tags para Formatação Específica:

  • sub: Exibe o texto como subscrito;
  • sup: Exibe o texto como sobrescrito.

  • Os Atributos das Tags

Os atributos em HTML são como adjetivos que fornecem informações adicionais sobre as tags, permitindo que você customize e refine a aparência e o comportamento dos elementos em sua página web. Eles são como chaves que abrem portas para funcionalidades específicas e permitem que você adapte o conteúdo às suas necessidades.

Imagine uma tag como um recipiente. Os atributos são como rótulos que você cola nesse recipiente para descrever o que ele contém e como ele deve ser usado.

  • id: Atribui um identificador único a um elemento;
  • class: Atribui uma classe a um elemento para aplicar estilos CSS;
  • href: Define o destino de um link;
  • src: Especifica a fonte de um recurso (imagem, vídeo, áudio);
  • alt: Fornece um texto alternativo para elementos como imagens;
  • style: Define estilos CSS diretamente na Tag (não recomendado para grandes projetos).
